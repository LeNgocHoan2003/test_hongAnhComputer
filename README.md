# test_honganhcom

A Flutter application for the `honganhcom` project.

## Prerequisites

Before running the project, make sure you have:

- [Flutter SDK](https://docs.flutter.dev/get-started/install)
- A device emulator (Android/iOS) or a physical device
- An editor such as Cursor, VS Code, or Android Studio

## Getting Started

1. Install dependencies:

```bash
flutter pub get
```

2. Run the app:

```bash
flutter run
```

## Project Structure

The project uses a feature-based structure in `lib/`, for example:

- `lib/features/`: Feature modules (presentation, logic, data)
- `lib/main.dart`: Application entry point
